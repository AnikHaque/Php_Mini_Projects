# PHP Mini Projects

This repository contains 4 beginner-level PHP mini projects for practicing core PHP concepts like conditionals, arithmetic operations, and basic authentication.

---

## 📁 Project List

### 1. Number Classifier
**Description:**  
Classifies a given number as Positive/Negative and Even/Odd.

**Sample Output:**  
`The number -4 is Negative and Even.`

---

### 2. Temperature Converter
**Description:**  
Converts temperature from Celsius to Fahrenheit and vice versa using formulas:

- Celsius to Fahrenheit: `(C × 9/5) + 32`
- Fahrenheit to Celsius: `(F - 32) × 5/9`

**Sample Output:**  
`25°C is equal to 77°F`  
`77°F is equal to 25°C`

---

### 3. Basic Auth System
**Description:**  
A basic login system using hardcoded username and password.

**Credentials:**  
- Username: `admin`  
- Password: `1234`

**Sample Output:**  
`Login Successful` or `Invalid username or password`

---

### 4. Simple Electricity Bill Calculator
**Description:**  
Calculates the electricity bill based on the number of units consumed:

| Units Consumed     | Rate per Unit (₹) |
|--------------------|-------------------|
| 0 – 50             | 3.50              |
| 51 – 100           | 4.00              |
| 101 – 200          | 5.20              |
| 201 and above      | 6.50              |

**Sample Output:**  
`Total electricity bill for 250 units is: ₹1180.00`
